Порядок установки:
1) Установить CRRuntime_32bit_13_0_17.msi (или CRRuntime_64bit_13_0_17.msi если windows 64-битная)
2) Установить visa503runtime.exe (это самораспаковывающийся архив)
3) Установить SetupKipTM.msi

При установке всех пакетов следует действовать в всоответствие с инструкциями Мастера установки
